# Introducción

El desafio de este proyecto consiste en simular el ticket de una empresa dedicada a llevar turistas desde su aeropuerto de llegada a su hotel de destino. La clase principal viene creada y solo hay que elaborar las otras clases y métodos que pide. La herramientas utilizadas serán el lenguaje de programación java, el software de control de versiones git y el gestor de proyectos gradle.
